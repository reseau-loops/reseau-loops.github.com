Les widgets interactifs sont essentiels à l’écosystème de Jupyter. Ils permettent de créer de l’interaction entre l’utilisateur et ses données dans l’interface du Notebook.

Les widgets sont des objets Python que l’utilisateur peut instancier et afficher dans le Notebook. Contrairement aux autres objets Python, ce n’est pas une représentation en chaîne de caractères qui s’affiche dans le Notebook mais un élément d’interface graphique : bouton, slider, carte 2D, graphique 3D etc.

Durant cette présentation, nous allons voir comment les widgets sont créés dans le Notebook et comment le noyau Python communique avec l’interface web du navigateur pour rendre ces widgets interactifs. Nous allons ensuite faire le tour de plusieurs modules de widgets comme ipyleaflet et ipygany.
